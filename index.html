CE QUE JE FAIS

Front-end

Le développement web frontal (aussi appelé front-end en anglais) correspond aux productions HTML, CSS et JavaScript d’une page internet ou d’une application qu’un utilisateur peut voir et avec lesquelles il peut interagir directement.

Back-end

Le Back-End, c’est la partie du code qui est exécutée par le serveur, il s’agît du travail qu’il réalise sur les pages Web des sites dynamiques avant de les envoyer au client.

CMS

WordPress est un outil de gestion de contenu (ou CMS) gratuit et Open Source parmi les plus populaires. Il est connu pour sa souplesse, grâce à des "thèmes" de mise en page (ou templates), extensions (ou plugins) et widgets très faciles à installer, permettant de pousser loin et aisément la personnalisation du site.
